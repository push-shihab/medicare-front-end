# MediCare Connect

## 📌 Project Name

**MediCare Connect**

## 📖 Description

MediCare Connect is a comprehensive healthcare management web application that connects patients with doctors and hospitals through a centralized platform. Built with a clean, minimal, Apple-inspired design, it provides a seamless experience for patients to book appointments, doctors to manage their schedules and prescriptions, and administrators to oversee the entire platform. The application features role-based dashboards, secure authentication, real-time analytics, and a user-friendly interface that makes healthcare accessible to everyone.

## ✨ Features

1. Users can register and login as Patient, Doctor, or Admin.

2. Protected routes only for valid authenticated users.

3. Login with Google authentication.

4. Patients can browse and search from 500+ verified doctors.

5. Patients can book appointments with their preferred doctors and time slots.

6. Patients can view their appointment history, payment records, and reviews.

7. Doctors can manage their schedule, availability, and consultation fees.

8. Doctors can accept or reject appointment requests from patients.

9. Doctors can create and manage digital prescriptions for patients.

10. Admin can verify or reject doctor registrations.

11. Admin can manage all users (patients and doctors) with suspend/delete options.

12. Admin dashboard with comprehensive analytics (appointments, revenue, user distribution).

13. Real-time statistics tracking (total doctors, patients, appointments, revenue).

14. Secure payment processing with Stripe integration.

15. User's data is encrypted securely in the MongoDB Database.

16. Responsive design that works seamlessly on mobile, tablet, and desktop.

17. Clean, minimal UI with consistent design system across all pages.

18. Role-based access control with separate dashboards for each user type.

19. Digital prescription management system.

20. Review and rating system for doctors.

## 📱 User Roles

### Patient

- Browse and search doctors
- Book appointments
- View appointment history
- Manage payments
- Write reviews
- Update profile

### Doctor

- Manage schedule and availability
- Accept/reject appointments
- Create prescriptions
- View patient reviews
- Update profile and qualifications

### Admin

- Verify/reject doctor registrations
- Manage all users
- View platform analytics
- Monitor appointments and payments
- Generate reports
